# Command:

Response list in table:
**_node index.js --action list_**

Get contact by id:
**_node index.js --action get --id 5_**

Add contact:
**_node index.js --action add --name Mango --email mango@gmail.com --phone 322-22-22_**

Remove contact:
**_node index.js --action remove --id 3_**
